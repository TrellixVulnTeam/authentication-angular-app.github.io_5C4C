Authentication App

Project test for authenticated users with Firebase backend API.

The app was authenticated by email and password and also communicated with token that generate from Firebase authenticate API.

What this app can do?

  • Sign up new users.
  • Logged in user from database.
  • List how many users have been sign up.
  • Shows the online user's status in real-time.
  • Shows date when they had created their accounts.
  • Shows alert error messages on user-interface
  • Auto login/logout by checking expired date account.
